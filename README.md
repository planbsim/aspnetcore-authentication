# Security Identity

A simple ASP.NET Core Identity solution which provides Authentication, Authorization and Facebook Login as external provider.

### Prerequisites

```
.NET Core Environment (ASP.NET Core 2.0, SQL Server-Connection)
```

```
Facebook APP
```
